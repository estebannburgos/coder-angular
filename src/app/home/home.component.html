Curso Angular

En este curso aprenderemos a crear un sitio web basado en el framework Angular. Nos sumergiremos en las mejores prácticas del diseño web, iniciando tu trabajo con HTML y CSS. Además de programar con Javascript para agregarle dinamísmo los sitios webs. Angular permitirá trabajar aplicaciones web con HTML y TypeScript pudiendo generar desde el prototipo hasta la implementación completa de una aplicación.
